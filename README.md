The Chosen
==========

A common class/interface/trait library for PHP 5.4+,
which consists of several classes/interfaces/traits chosen by myself for my personal use.

By the way, *"Chōsen"* (ja: 挑戦) is a word meaning *"Challenge(s)"* :)
